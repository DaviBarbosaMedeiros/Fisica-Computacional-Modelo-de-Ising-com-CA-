# Fisica-Computacional-Modelo-de-Ising-com-CA-
Este é um repositório feito para a cadeira de Física Computacional, onde apresentamos um código em python para o modelo de Ising com o uso de células automatas. Esse é um trabalho desenvolvido por Davi Barbosa de Mederios, Haroldo Abreu de Castro Filho e Kawan de Brito Bezerra, todos do curso de Física de Materiais da Universidade de Pernambuco.
